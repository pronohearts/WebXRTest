# Nippo Web (脱Python・Webアプリ版)

このフォルダは、Python をインストールせずにブラウザだけで動く「日報管理システム」の Web アプリ版です。

## 🚀 使い方
1. このフォルダの中にある `index.html` をダブルクリックしてブラウザで開きます。
2. 他に必要な準備（Pythonのインストールなど）は**一切不要**です。

## 📁 フォルダ構成
- `index.html` : アプリの本体（ここをダブルクリック）
- `css/style.css` : デザイン（プレミアム・ガラスモーフィズム）
- `js/app.js` : アプリの動作ロジック
- `js/gsheets.js` : Google スプレッドシート連携用

## 📊 データの保存について
- **デフォルト（ローカル保存）**: ブラウザの保存領域 (LocalStorage) にデータが保存されます。自分一人の PC で使う場合に適しています。
- **チーム共有 (Google Sheets)**:
  `js/gsheets.js` の中に記載されている手順に従って、Google Apps Script を設定することで、複数人で同じスプレッドシートにデータを集計できるようになります。

## 🌐 他の人への配布方法
1. `web` フォルダをそのまま ZIP で圧縮して相手に送ります。
2. 相手に「解凍して `index.html` を開いて」と伝えるだけです。
3. もしサーバー（GitHub Pages など）をお持ちであれば、この中身をアップロードするだけで、世界中どこからでもアクセスできるようになります。
